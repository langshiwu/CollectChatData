# CollectChatData
CollectChatData a python project collectting wechat messages with itchat module  
The data collectted can make up corpus for a chatBot  
构建数据库，运行collectData.py后扫码登陆微信，可以将微信的聊天记录存于数据库中，作为对话Bot的训练语料。
